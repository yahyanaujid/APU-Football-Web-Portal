# APU Football Club Web Portal

A comprehensive, 20-page responsive web platform built for the **Asia Pacific University (APU) Football Club** to centralize match updates, team management, and member registration.

## 🛠️ Technical Stack
* Languages: Semantic HTML5 and CSS3.
* Interactivity: Vanilla JavaScript (ES6) for dynamic components.
* Design Methodology: Wireframing, Site Mapping, and Storyboarding.

## 🚀 Key Features
* Interactive Components: JavaScript-powered countdown timers, Player of the Season voting polls, and image flip-cards.
* Operational Management: Dynamic event calendars, match highlight galleries, and validated membership registration forms.
* Unified Branding: Implementation of a centralized Master Stylesheet (`master.css`) to ensure consistent typography and layout across all 20 pages.

## 👔 My Role: Technical Integration Lead
As the primary lead for the project's technical strategy and documentation, I was responsible for the following:

* System Standardization: Authored 70% of the technical discussion and troubleshooting strategy, resolving critical integration conflicts during the final build.
* Code Quality & Accessibility: Audited the project to replace deprecated tags with modern Semantic HTML5 elements (e.g., `<nav>`, `<section>`), improving browser compatibility and accessibility.
* Architectural Fixes: Established a uniform naming convention and shared folder structure to resolve broken internal links and directory confusion.
* Feature Coordination: Led technical review meetings to identify and eliminate redundant work, such as overlapping calendar features.

## 🧠 Challenges & Solutions
* The Consistency Challenge: With four developers working independently, the site's initial design was fragmented.
* Solution: I helped implement a shared layout template and master stylesheet to provide a professional, unified "one-brand" user experience.
* Asset Optimization: Large media files were offloaded to external hosting to keep the repository lightweight and follow modern web performance standards.
